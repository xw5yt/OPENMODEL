OPENMODEL is a powerful AI terminal assistant that can execute commands on your computer.
It uses OpenRouter API to generate responses and can run any command you ask it to.
Just type your command and let OPENMODEL do the rest.
OPENMODEL have integrated portable python, and you dont need
installed python for start OPENMODEL.
Remember, with great power comes great responsibility - be careful what you ask it to do!

==========================================================================================

Example Commands:
- List files: "dir" or "!ls"
- Change directory: "cd foldername"
- Run Python file: "python script.py"
- Clear screen: "!clear"
- Ask AI to create a file: "create file.txt with content 'Hello World'"

- Always double-check commands before running
- Avoid running scripts from untrusted sources
